# Bank Account System in C

## Overview

This project is a simple console-based bank account management system implemented in C. It allows users to perform basic banking operations such as deposit, withdrawal, and balance checking.

## Features

* Deposit money into account
* Withdraw money from account
* Check account balance
* Basic account creation
* Menu-driven interface

## Concepts Used

* Functions
* Pointers
* Conditional statements (if-else)
* Loops (while)
* User input handling (scanf, fgets)

## How to Run

1. Compile the code using a C compiler (GCC)

   gcc bank.c -o bank

2. Run the program
   
   ./bank
   
3. Follow the menu instructions

## Code Explanation

* Functions are used for deposit, withdrawal, and balance checking
* Pointers are used to update the bank balance
* Loop is used to continuously perform operations
* Input is taken from user using standard input functions

## Applications

* Learning C programming basics
* Understanding pointers and functions
* Building logic for real-world problems

## Limitations

* No file storage (data not saved permanently)
* Single user support
* Basic validation only
